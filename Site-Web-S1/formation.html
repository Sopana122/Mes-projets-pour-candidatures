Présentation :

L'academie SVS, propose plusieurs diplomes avec des formations qui dure entre 3 à 20 ans. Les diplomes sont concus pour que les personnes, élèves et étudiant puissent s'adapter le plus possible, nottament avec le diplome BA (Baby Agent). Nous avons aussi des diplomes de voleur specialisé, agent-seccret, militaire ou encore acteur.

Vous trouverez ci-dessous les differentes formations ainsi que diplomes proposé par l'academie.

Formations AS :

La formation Agent Secret connu sous le nom de AS est une formation qui consiste à acqueriur les qualités nécessaires pour être un agent secret. Le reponsable de cette dernière est Mr LUPIN Arsene, le meilleurs espion du monde. Il a été classer 1er du classemnt 5 ans de suite. Son rôle est non seulement de former les meilleurs agents au monde afin de servir notre nation mais aussi de transmettre ses connaisances à nos futurs espions.

Modules :
- Opération de cyber-renseignement et lutter contre les cyber-attaques;
- Traduction de conversation et des documents;
- Activitée de survies avec specilisation dans un sport de combat;
- Cours géographique pour comprendre l'environnement qui nous entoure;

PRIX : 55 050 $

Formations BA :

La formation Baby Agent connu sous le nom de BA est une formation qui consiste à acquerur les qualités nécessaires pour être un baby agent.

Modules :
- Se spécialiser dans un sport de combat pour le self defence
- Apprentissage des bases en informatique
- Activités manuelles afin de developper notre esprit

PRIX : 35 000 $

La formation Voleur SPE connu sous le nom de VS est une formation qui consiste à acqueriur les qualités nécessaires pour être un voleur spécialisé.
